# Security Policy

## Supported Versions

Only the latest version of this portfolio repository is supported. Security updates and patches will be applied directly to the `main` branch.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please do not open a public issue. Instead, report it privately to the maintainer:

* **Email**: [Saravanashrutheesh@gmail.com](mailto:Saravanashrutheesh@gmail.com)

We aim to acknowledge your report within 48 hours and provide a resolution timeline shortly after.
